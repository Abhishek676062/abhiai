# Google Business Profiles

Place your GBP screenshots or badges here. Supported formats: PNG, JPG, WebP

## Configuration
Update the GBP links in: `src/components/sections/CredentialsSection.tsx`

Each profile entry needs:
- `name`: Business/profile name
- `url`: Google Business Profile URL
- `image`: Optional screenshot path (e.g., "/showcase/gbp/profile-1.png")
- `rating`: Star rating (e.g., 4.8)
- `reviews`: Number of reviews
