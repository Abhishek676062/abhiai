# AI Generated Images

Place your AI-generated showcase images here. Supported formats: PNG, JPG, WebP

## File Naming Convention
- `ai-image-1.png` — First AI generated image
- `ai-image-2.png` — Second AI generated image
- etc.

## Configuration
Update the image data in: `src/components/sections/AIGallerySection.tsx`

Each image entry needs:
- `src`: filename (e.g., "/showcase/images/ai-image-1.png")
- `alt`: Description for accessibility/SEO
- `title`: Display title (optional)
