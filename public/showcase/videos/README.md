# AI Video Showcase

Place your AI demo videos here. Supported formats: MP4, WebM

## File Naming Convention
- `video-1.mp4` — First AI demo video
- `video-2.mp4` — Second AI demo video
- `video-3.mp4` — Third AI demo video

## Configuration
Update the video data in: `src/components/sections/VideoShowcaseSection.tsx`

Each video entry needs:
- `src`: filename (e.g., "/showcase/videos/video-1.mp4")
- `title`: Video title
- `description`: Short description
- `thumbnail`: Optional thumbnail image path
